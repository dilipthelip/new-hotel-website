### How to run a Docker Image (Docker Container = Instance of a Docker Image) ?
#### Approach1:
-it -> stands for interactive mode.
```
docker run -it -p 8080:8080 --name lastmile [imagename]
````
#### Approach2:
-dt -> This launches the image and runs in the background.
```
docker run -dt -p 8080:8080 --name lastmile [imagename]
````
Command - **docker run -p 8090:8080 -t lastmileapi:1.0**
The above command will bring the app up.
8090:8080 -> 8090 is the port docker will host the app.
